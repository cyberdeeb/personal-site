Random Photo Generator

In this project I declare an array using JavaScript to store the images. I then declare a JavaScript variable to store a random value calculated using Math.floor(Math.random()*photo.length) method. It will generate a random number between 0 and the length of the array that will be assigned to the images to display randomly. The above steps are place in a user-defined function (getRandomImage), which will call by clicking on a Generate Image. In HTML code, I utilized the img tag and left src blank. Once the random number is calculated and the associated image is pulled, I use setAttribute to place the image name from the array into the src attribute of the img tag.
